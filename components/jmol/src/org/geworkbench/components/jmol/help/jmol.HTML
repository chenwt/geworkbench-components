JMOL

JMOL supports viewing 3D molecular structures of proteins, DNA and RNA which are loaded into geWorkbench as PDB format files. A full description of the capabilities and functionality of JMOL can be found at (http://jmol.sourceforge.net/ )

Viewing PDB format files

1. Load a PDB file into a Project folder.

2. Navigate to the JMOL image viewer.

3. Modify the display of the image as needed.
