JMOL

JMOL supports viewing 3D molecular structures of proteins, DNA and RNA which are loaded into geWorkbench as PDB format files. A detail description of JMOL is an open-source project can be found at (http://jmol.sourceforge.net/ )

View molecular structure (PDB format) files

1. Load a PDB file into a Project.

2. Navigate to the JMOL image viewer.

3. Modify the display of the image. Operations include: select, render, color and label the molecule and its constituent parts.
