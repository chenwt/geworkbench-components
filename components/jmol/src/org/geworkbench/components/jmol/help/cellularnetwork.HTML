Cellular Network

This help section describes accessing protein interaction data residing in the Cellular Network Knowledge Base (CNKB). The CNKB is a repository of interactions between pairs of proteins (these interactions can be either computationally or experimentally derived). Both direct, physical interactions can be captured as well as indirect transcriptional relationships (where an interaction is between a transcription factor and its gene target). Each pairwise interaction has an associated confidence indicator (a value between 0 and 1) reflecting the strength of the underlying data, whether experimental or computational.

The Cellular Network visualization is composed of the following three sections:

1. Activated Marker List:

Marker Name: The marker name appears Italized to indicate interaction information has not been retrieved and bold face indicates the interaction information has been retrieved.

Gene Name: This is corresponding to the marker, if known; empty otherwise.

Gene Type (derived from the GO annotation):

。 TF : Transcription Factor

。 K: Kinase

。 P: Phosphatase

2. Selected Marker List: Lists the subset of activated markers selected from the Activated Marker List.

3. Throttle Graph: Clicking on Preview maps the interactions selected. The graph allows users to throttle (for the genes in the Selected Markers table) which interactions to work with using as a criterion the interactions’ confidence indicator

Retrieve CNKB interaction sets

1. Create an active marker set in the Selection Area of the application (lower left pane). The Activated Marker List is updated to reflect the markers set.

2. Select one or more markers from the Activated Markers List by double clicking on Marker rows. The Selected Marker List is updated to reflect the markers selected from the Activated Markers List.

3. Indicate interactions of interest using the checkboxes on each row or the ‘All Protein – DNA’, ‘All Protein-Protein’ checkboxes. Display Preference is used to control how many of the columns listed above they want visible within the Selected Markers List.

4. Click Refresh.

. Selected Markers List: The system retrieves from CNKB the interaction sets for all genes in the Selected Markers list. The system updates “#of Prot-Prot” and “# of Prot-DNA” columns within the Selected Markers table.

. Throttle Graph: Users can select a value for the X-axis by using either the graph slider or Threshold box. Modifying the values of X- axis updated the contents of the Selected Markers table “Prot-Prot #” and “Prot-DNA” columns.

5. Click Create Network. The system builds an adjacency matrix comprising those interactions that were used to compute the contents of non-grayed out cells of the “Prot-Prot #” and “Prot-DNA” columns in the Selected Markers table. The adjacency matrix is placed in the Project Folders component under its parent microarray dataset and can be viewed in the Cytoscape visualization.
