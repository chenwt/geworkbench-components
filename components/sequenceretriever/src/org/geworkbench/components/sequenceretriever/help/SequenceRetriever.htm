Sequence Retriever

The Sequencer Retriever component retrieves nucleotide or protein sequences for selected markers (genes).

  • Nucleotide sequences are obtained from the UC Santa Cruz server. This is genomic sequence and is retrieved relative to the gene transcription start site. The user can enter how many bases up- and down-stream from the start site are to be retrieved.
  • Protein sequences are obtained from the European Bioinformatics Institute (EBI) server.

 

Controls

The major controls in the Sequence Retriever component are:

Type: The type of sequence to retrieve, either DNA or Protein. The location from which the sequence will be retrieved is shown on a separate pulldown.

Get Sequence: Sequences for the selected sequences will be retrieved (if available).

Add to Project: Retrieved sequences can be add as a new dataset in the Project Folders component. This allows their reuse by other components such as BLAST or Promoter.

Sequence Extent (- or +). For nucleotide sequences, the distance up (-) and downstream (+) of the transcription start site can be specified.

View: Line or Full Sequence. Switches between a simple line representation and a detailed sequence listing.

 

Preliminaries

The Sequence Retriever component uses marker name to gene name mappings. These are only available if the annotation file for a microarray platfrom was read in at the time that the dataset was loaded.

Before retrieving sequences, a set of markers must be defined and activated in the Markers component:

 

Retrieving sequences:

The Sequence Retriever component displays any marker sets activated in the Markers component:

 

Select the type of sequence desired, DNA or Protein. Currently only one source for each sequence type is provided.

To retrieve sequences for the desired markers, click Get Sequence.

A pop-up appears showing the available genome assemblies. Confirm that the desired assembly is selected and click OK.

 

 

The retrieved sequences are displayed. For DNA, the transcription start site is show as the boundary between blue and red segments.

 

Double-click on any of the line representations to see the detailed sequence. Double-clicking again returns you to the line view.

 

Adding sequences to a new dataset:

In the line view, one can select sequences to be added to a new dataset in the Project Folders component.

 

Clicking on Add to Project will add them. A popup appears asking for a name for the new dataset.

 

 

After the name is entered, the new dataset appears in the Project Folders component.
