Scatter Plot

The Scatter Plot component allows the visual comparison of the expression data for pairs of microarrays or markers. Expression data is plotted with one marker or microarray on the x-axis and the other on the y-axis. Up to 6 plots can be viewed at once. Each plot has the same marker/microarray on the x-axis. Basic Usage

The steps of basic usage are indicated with numbers in the below screenshot.

  1. This tab switches between Marker/Marker and Array/Array plots. Switching between these tabs does not affect the charts in the other tab.
  2. Markers or arrays can be searched interactively using this control.
  3. The first marker or array selected is placed on the x-axis and his highlighted in black. A different marker or array can be placed on the x-axis by right-clicking the marker/array name and choosing Put on X-Axis.
  4. Subsequent selections of markers or arrays after a marker/array is on the x-axis results in the creation of a chart. Plotted markers/arrays are highlighted in grey. Clicking again on one of these markers/arrays results in the plot being removed.
  5. Clicking the Rank Statistics Plot checkbox transforms the data for analysis. The x and y values are sorted and plotted according to their rank.
  6. By default, a black reference line with slope 1 is displayed in each chart. This may be turned off with the Reference Line checkbox. Also, the slope of the line may be adjusted in the Slope textbox.
  7. The Clear Charts button removes all charts and removes the x-axis selection. The Print button prints the charts after allowing the user to adjust the page setup and choose a printer. The Image Snapshot button captures the charts as an image and places it in to the project underneath the current data set.

Each chart can be manipulated by right-clicking anywhere in the plot area. This brings up a menu that allows the chart to be invidually saved as an image or printed, zoomed and visual properties adjusted.

Panel Selections

Markers or microarrays that are members of active panels will be plotted with unique visual properties. These panel selections are managed for arrays and markers in the Phenotype Panel and the Gene Panel, respectively. Consider an example where the two panels are activated in the Phenotype Panel:

Two phenotypic panels BL and FL are activated in the Phenotype Panel. This results in the arrays in these panels to take on different visual properties in the charts:

Unchecking the All Arrays checkbox (indicated above with a 1) will remove all arrays from the charts except those that are in activated panels. So, in the above screenshot, all the red squares would be removed from the chart. Corresponding behavior can be achieved with marker plots, the Gene Panel and the All Markers checkbox.

The visual properties of a panel may be adjusted. From within the Phenotype or Gene Panel, right-click a panel and choose Change Visual Properties.

A dialog opens that allows the shape and color to be changed for that panel. These properties are honored in the Scatter Plot as well as other caWorkbench components.
