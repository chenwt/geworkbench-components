Microarray View

The Microarray View is located in the View Area in the top right of the application. Measurements in the selected microarray dataset are displayed as spots over a green-red color spectrum in the Microarray View. For datasets comprising more than one microarray, the first microarray data is shown as default, use the slider below the display area to view other microarrays. Missing data is separately color coded (see Missing Values Filter in Filtering Panel for additional information).

Basic Usage

The steps of basic usage are indicated in the below screenshot.

A. All Arrays: The checked/ selected All Arrays checkbox indicates the component is displaying the entire set of arrays from the loaded dataset. The user can clear the All Arrays check box to display only arrays in the activated Phenotype Panel.

B. All Markers: The checked/ selected All markers checkbox indicated the component is displaying the entire set of markers from the data. To include only gene panels that are currently activated, the user clears the checkbox. If cleared, the markers displayed are marker drawn corresponds to markers in the activated Marker panels.

C. The name of the array displayed in the component.

D. Intensity slider: This widget increases or decreases the intensity of the display.

E. Microarray slider: This widget is used to scroll through the Microarrays in the currently selected dataset. The grid is updated with spots representing marker values in the Microarray being displayed. The relative positions of the markers remain the same across all the Microarray grids.

F. Clicking on graph: Clicking on any of the grid cells in the microarray display highlights the marker clicked in the Gene Panel.

Right click on graph: This displays a pop-up menu with the following options:

Show marker will highlight the selected marker(s).

Remove marker will remove the selected marker and repaint it with original color.

Image Snapshot saves the drawn grid image as a child of the currently selected dataset
