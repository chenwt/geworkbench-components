Microarray View

To view a dataset using Microarray View, first select a dataset in the "Project Folder" pane and then select the "Microarray Panel" option in the main display pane at the top-right section of the interface. Measurements in the selected microarray dataset will be displayed as spots over a green-red color spectrum. Missing data are separately color coded. For datasets comprising more than one microarray, only the first microarray is shown. Use the slider below the display area to view other microarrays.

This component contains some tools for users to manipulate the microarray display:

  • Button with bulb: Widget enables user to toggle MouseOver information on the grid.

  • Right Click on graph: This shows a pop-up menu with a 'Save Image' option. Selecting this option lets user save the drawn grid image as a child of the currently selected dataset

  • Clicking on graph: Clicking on any of the grid cells in the microarray display highlights the marker clicked in the Marker Panel.

  • Show Phenotypes: Widget lets user subselect Microarrays to view. The number of points on the Microarray slider equals the number of Microarrays in actrivated Phenotype panels.

  • Show Panels: Widget lets user subselect number of markers to view. The marker drawn correspond to markers in the activated Marker panels.

  • Microarray slider: Widget lets user walk through all the Microarrays in the currently selected dataset. The grid is updates with spots representing marker values in the Microarray being displayed. The relative positions of the markers remain the same across all the Microarray grids.
