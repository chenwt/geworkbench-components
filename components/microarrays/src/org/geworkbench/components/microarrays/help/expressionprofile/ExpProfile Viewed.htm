Expression Profile View

The Expression Profile is located in the View Area (see image below) in the top-right of the application.  The Expression Profile view makes it possible to visualize changes in the gene expression levels across different hybridizations. This is useful especially in the analysis of time course or dose response experiments. Since the tool generates a graphical representation of relative expression levels across two or more arrays, the Expression Profile view can only be applied to datasets containing multiple arrays. 

Measurements in the selected microarray dataset are displayed in the graph with corresponding x-y axes coordinates. Each Marker is displayed as a unique colored line. When the mouse is scrolled over a data point on the chart, the tool tip displays a description of the data point.
