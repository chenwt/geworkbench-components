Expression Profile View

To view a dataset using Expression Profile View, first select a dataset in the "Project Folder" pane and then select the "Expression Profile" option in the main display pane at the top-right section of the interface. Measurements in the selected microarray dataset will be displayed in a graph with corresponding x-y axes coordinates. Measurements for the same marker are connected by straight line between adjacent microarrays.

This component contains some tools for users to manipulate the graph display:

  • Button with bulb: Widget enables user to toggle MouseOver information on the graph.

  • Right Click on graph: This shows a pop-up menu with a 'Save Image' option. Selecting this option lets user save the drawn graph image as a child of the currently selected dataset

  • Clicking on graph: Clicking on any of the profiles in the graph highlights the marker clicked in the Marker Panel.

  • Show Phenotypes: Widget lets user subselect profile samples to view. The number of grid points for each marker profile equals the number of Microarrays in activated Phenotype panels.

  • Show Panels: Widget lets user subselect number of profiles to view. The marker profiles drawn correspond to markers in the activated Marker panels.

  • Mean Profile: Widget lets user view mean marker profiles of either all markers in the selected dataset or a subselection of markers in the Marker panels. Appropriate variance bars are also drawn.
