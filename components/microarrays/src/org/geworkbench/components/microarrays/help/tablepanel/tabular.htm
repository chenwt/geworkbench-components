Tabular Microarray Panel

This Visual Area component displays a spreadsheet view of t the markers values in a microarray set. For sets involving multiple arrays (as in the figure below), microarrays are displayed across columns and markers across rows.

To view a dataset using Tabular Microarray Panel

。           Select a dataset in the Project Panel and then select the Tabular in the View Area at the top-right section of the interface.

Navigating Tabular Microarray Panel View

。           Change the width of columns by using the mouse by dragging the boundary on the right side of the column heading until the column is the width that you want.

。           Table cells corresponding to markers with missing values are displayed color-coded in yellow.

。           The amount of data displayed can be controlled by the following check boxes at the bottom of the component:

Activated Arrays:  When this checkbox is selected, the microarrays displayed are included in an activated phenotypic panel. Otherwise, data for all microarrays in the selected set are shown.

Activated Markers: Similarly, selecting this checkbox results in the display of markers that are part of activated marker panels  Otherwise, data from all markers are shown.
