Expression Value Distribution

The Expression Value Distribution (EVD) is located in the View Area in the top-right of the application. This component uses a line plot to graphically summarize the distribution of expression values within a microarray. The x axis represents the value of the range and the y axis the occurrence of markers with expression value in that range. For each array, the marker value frequencies are plotted over the range [m, M], where m, M are respectively the smallest and highest expression values observed across all arrays of a microarray set. Navigating an EVD

The steps of basic usage are indicated with numbers in the below screenshot.

1. Bin size and total bans: Each expression range on the EVD corresponds to a “bin”. The size of each bin and total bans are displayed at the upper left corner of the component. The number of total bins equals (m – M)/ number of bin size.

2. Tooltip: When a user scrolls over a plot, the enabled tool tip displays the name of the array. The tooltip display can be disabled by selecting the tooltip icon. The disabled tooltip icon appears grayed out.

3. Change the base array: The slider can be used to navigate through the microarrays in the selected dataset. As the user moves the slider to the next tick which represents a microarray, the grid is updated with positions representing the marker values in the Microarray being displayed. The base array always labels with the red color.

4. Activated Marker and Array Checkbox: If a microarray or gene panel is created and activated, the EVD reflect plots for each array in the panels when the checkbox is selected (default). Each array in the activated panel is displayed on the graph and its corresponding plot points are linked with lines, while the points corresponding to the base array are unconnected. Unselecting the checkbox results in data from all microarrays/marker in the selected set are shown.

5. One Color per Array: The EVD lines and plots appear color-coded, based on the color preferences of the arrays panel. All the arrays in a panel are shown with the same color. The user can modify the color display to reflect a unique color per array by selected the checkbox One Color per Array.

6. Legend: A legend appears at the bottom of the plot indicating which color and shape corresponds to which panel. Modification of image display preferences are described in details in Preferences. The base array always list as the first one.

7. Image Snapshot: Clicking on this button creates image snapshots that are saved in the project.

8. Zoom: Zooming in can be down by left clicking and drag down over a region of the image or selecting right clicks Zoom In option. Zooming out, left clicks and drag upward or right click and select Zoom Out option.

9. T-Test: This button computes a t-test statistic for each marker and updates the graph title from EVD to T-Test and disabled the base array slider. Case and control panels must be created and classified (Marker Panel Help section for additional information).

10. Print: Prints the displayed EVD. The print dialog pop up is displayed to support printer selection.

11. Selected Values from/to: The starting/ending X axis location that allow markers to be selected graphically and filtered. The number of current selected genes is displayed next to the Total bins. “Add to Panel” button adds the selected genes to the Gene Panel.
