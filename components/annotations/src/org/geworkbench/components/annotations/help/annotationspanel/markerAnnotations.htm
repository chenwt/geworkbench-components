Marker Annotation and Pathway Listing

Marker Annotations is a visualization component located in the Visual Area in the top right of the application. This is a utility for displaying annotations about genes and their respective biochemical pathways. This information is retrieved via the Cancer Bioinformatics Infrastructure Objects (caBIO) services offered by the National Cancer Institute.

Basic Usage

This component contains the following tools to probe the pathways:

· Retrieve Annotation: Triggers a query to caBIO servers based on markers existing in activated Marker panels.

· Clear: Clears results of caBIO query.

To Display Annotation

1. Create an activated marker panel with markers of interest.

2. In the Marker Annotation visualization component, select Retrieve Annotation. The component composes a list of genes corresponding to those markers in the activated Marker Panels and retrieves the Biochemical Pathways each such gene is known to participate in.

To access annotations

。 Click on a gene of interest and the system will return an annotation from the NCI CGap database on your default browser.

To access the pathway diagrams

。 Click on the pathway name link of in and the pathway diagrams (published by KEGG (http://www.genome.ad.jp/kegg/) and BioCarta (http://www.biocarta.com ) are displayed in the Pathway Panel in the caBIO Pathways tab.
