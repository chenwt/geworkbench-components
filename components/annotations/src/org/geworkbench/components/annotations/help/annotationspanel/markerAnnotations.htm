Marker Annotation and Pathway listing

This is a utility for displaying annotations about genes and their respective biochemical pathways (information retrieved via the Cancer Bioinformatics Infrastructure Objects (caBIO) services offered by the National Cancer Institute). This component composes a list of genes corresponding to the markers in activated Marker Panels and retrieves a list of the Biochemical Pathways each such gene is known to participate in. The Pathway diagrams can be visualized by the Pathway Panel (caBIO Pathways)

This component contains some tools for users to probe the pathways:

  • Use Panels: This widget triggers a queries to caBIO servers based on markers existing in activated Marker panels.

  • Clicking on markers on the left column: Clicking the marker entry in the left column causes annotation for that gene to be bought up ina browser. The annotation for the gene is obtained from NCICB's cGAP project
