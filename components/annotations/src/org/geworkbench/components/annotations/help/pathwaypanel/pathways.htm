Pathway

This is a utility for displaying biochemical pathways information as retrieved from Cancer Bioinformatics Infrastructure Objects (caBIO) services offered by the National Cancer Institute. The application retrieves this information using the Annotations Panel and displays the pathways in this component.

To access pathway diagrams

1. Create an activated marker panel.

2. In the Marker Annotation visualization component, select Retrieve annotations. The component composes a list of genes corresponding to those markers in the activated Marker Panels and retrieves the Biochemical Pathways each such gene is known to participate in.

3. Click on the pathway name link of interest.

4. The pathway diagrams are displayed in the Pathway Panel in the caBIO Pathways tab. The diagrams are published by KEGG (http://www.genome.ad.jp/kegg/ ) and BioCarta (http://www.biocarta.com )

Note: When the mouse moves over any of the markers displayed in the pathway diagram the cursor changes to a 'hand' cursor. Clicking the marker at this point causes annotation for that gene to be bought up in a browser.
