Biochemical Pathway Visualization

This is a utility for displaying biochemical pathways information as retrieved from Cancer Bioinformatics Infrastructure Objects (caBIO) services offered by the National Cancer Institute. The application retrieves this information using the Annotations Panel and displays the pathways in this component.

This component contains some tools for users to probe the pathways:

  • Clicking on markers in the pathway diagram: When the mouse moves over any of the markers displayed in the pathway diagram the cursor changes to a 'hand' cursor. Clicking the marker at this point causes annotation for that gene to be bought up ina browser. The annotation for the gene is obtained from NCICB's cGAP project
