Mean-Variance Normalizer

For every marker profile, the mean measurement of the entire profile is subtracted from each measurement in the profile and the resulting value is divided by the profile’s deviation. Only non-missing values participate in the computation of the mean and deviation.

The parameters for this normalizer are:

  1. Missing values: Designates how to process any missing values in a marker profile, prior to centering them. Available options are:
    • Min profile: first normalizes all non-missing measurements for the marker corresponding to a missing value. Then, replaces the missing value with the minimum among the normalized values.
    • Max profile: first normalizes all non-missing measurements for the marker corresponding to a missing value. Then, replaces the missing value with the maximum among the normalized values.
    • Zero: replaces a missing value with 0.
    • Ignore: leaves missing values unchanged.
