Missing Values Filter

This filter discards from a microarray set all markers that have "missing" measurements in more than X microarrays (X is a user specified value). The markers that meet the filter criteria are removed from all the microarrays in the set.

In the example shown below, the filtering operation will remove a marker if the marker has a missing value in 10 or more microarrays ("Maximum number of missing arrays" designates the maximum number of arrays where we are willing to tolerate a missing value).

To Filter Missing Values

1 In the Filtering Panel, select the Missing Values Filter from the selection list.

2 Enter the missing measurements criteria.

3 Click on Filter. The View Area is updated to reflect the filter applied.

Note: Filtering is performed directly on the loaded dataset; a separate copy is not generated
