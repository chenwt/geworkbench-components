2 Channel Threshold Filter

This is a filter appropriate for treating 2-channel data (Genepix). It allows users to filter each channel signal separately. Similar to the expression threshold filter, users define reference ranges. Here, though, there are 2 reference ranges designated: one for the Cy3 channel (green) and one for the Cy5 channel (red). Marker measurement are set to "missing" if the intensity of either channel is outside (or inside) their corresponding range.

The parameters for this filter are:

  1. Cy3 Range Min, Cy3 Range Max: Real numbers that define a reference range [min, max] against which the green channel intensities will be compared .
  2. Cy5 Range Min, Cy5 Range Max: Real numbers that define a reference range [min, max] against which the red channel intensities will be compared .
  3. Filter values: Designates how the channel intensities are compared to the reference range. Available options are:
    • Inside range: Sets to "missing" the measurement for a marker if either channel intensity is inside its corresponding the reference range.
    • Outside range: Sets to "missing" the measurement for a marker if either channel intensity is outside its corresponding the reference range.

The intensity for a channel is calculated by subtracting the background measurement from the foreground measurement for the channel.
