Multi T-Test Analysis

This analysis performs pair-wise t-tests between multiple groups of phenotypes. The results are multiple-test corrected using Holm test correction. The Multi T-Test analysis should be used instead of T-Test if you plan to compare multiple groups. This analysis should be used instead of the regular t-test if you plan to compare multiple groups. The values of the Multi T-Test can be seen in the Volcano Plot and Color Mosaic.

Apply Multi T-Test

1 From the Analysis tools, select Multi T-Test Analysis from the selection list.

2 Update the parameters settings as described in the parameter section.

3 Click Analyze. The analysis results include the following:

。 The values of the Multi T-Test can be seen in the Volcano Plot and Color Mosaic created for each comparison.

。 The project panel is updated to reflect a child data node for each comparison.

。 In the Marker Set, the markers which met the significance test are included in groups for each pair of phenotypes compared.

Parameter Settings

The T-Test analysis parameters are as follows:

Critical P-Value: This is the p-value below which markers will be considered to be significant.

Compare Panels: Array/Phenotype Sets associated with the dataset are displayed in this section. Select the Array/Phenotype Set checkboxs to indicate the sets to include in the analysis. A minimum of two sets are required.

All Arrays: This has no effect in this analysis.

All Markers: Overrides any marker selection to display all markers are included in the T-Test.

Volcano Plot

A volcano plot is a scatter plot of fold change (log-2 difference) and significance. Markers that are far from the origin (0, 0) are generally considered to be candidates for further analysis.

Description

。 Plot Header: Lists the two marker groups compared.

。 Y Axis: -log10 significance. Each unit on the y-axis is an order of magnitude more significance, which higher values being more significant.

。 X Axis: A measure of differential expression such as a log-ratio. Note that data should be log-normalized prior to displaying a Volcano Plot.

。 Tool tip: The marker name, gene symbol and the significance score (p-value).

Right click options:

。 Zoom: Increases or decreases screen magnification.

。 Print: Opens the print dialog box to print the plot displayed.

。 Save As: Opens the Save As dialog, where you can: save the document as PNG to a local disk or a remote location.
