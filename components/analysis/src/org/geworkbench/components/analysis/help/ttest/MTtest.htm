Multi T-Test Analysis

This analysis performs pair-wise t-tests between multiple groups of phenotypes. The results are multiple-test corrected using Holm test correction. This analysis should be used instead of the regular t-test if you plan to compare multiple groups.

Apply Multi T-Test

1 From the Analysis Panel, select Multi T-Test Analysis from the selection list.

2 Update the parameters settings as described in the parameter section.

3 Click on Analyze. The analysis results:

。 The values of the Multi T-Test can be seen in the Volcano Plot and Color Mosaic panels created for each comparison.

。 The Project panel is updated to reflect a child data node for each comparison.

。 In the Marker Panel, the markers which met the significance test are included in panels for each pair of phenotypes compared.

Parameter Settings

The T Test analysis parameters are as follows:

n Critical P-Value

n Compare Panels

Critical P-Value

This is the p-value below which markers will be considered to be significant.

Compare Panels

Phenotype panels associated with the dataset are displayed in this section. Select the panel checkbox to indicate the panels to include in the analysis. A minimum of two panels are required.

All Arrays

This has no effect in this analysis

All Markers

This overrides any marker selection in the Marker Panel so that all markers are included in the t-test.

Volcano Plot

A volcano plot is a scatter plot of fold change (log-2 difference) and significance. Markers that are far from the origin (0, 0) are generally considered to be candidates for further analysis.

Description

§ Plot Header: Lists the two marker panels compared

§ Y Axis: -log10 significance. Each unit on the y-axis is an order of magnitude more significance, which higher values being more significant.

§ X axis: A measure of differential expression such as a log-ratio. Note that data should be log-normalized prior to displaying a Volcano Plot.

§ Tool tip: the marker name, gene symbol and the significance score (p-value).

§ Right click options:

o Zoom: Increase or decreases the screen magnification.

o Print: Open the print dialog box to prints the plot displayed.

o Save As: Opens the Save As dialog, where you can: save the document as PNG to a local disk or a remote location.
