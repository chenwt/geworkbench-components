Expression Threshold Filter

This filter sets to "missing" all marker measurement that are outside (or inside) a user defined range.

The parameters for this filter are:

Range Min, Range Max: These are real numbers that define a reference range [min, max] against which marker measurements will be compared.

Filter values: Designates how the marker measurements are compared to the reference range. Available options are:

o             Inside range: Sets to "missing" a measurement if that measurement is inside the reference range.

o             Outside range: Sets to "missing" a measurement if that measurement is outside the reference range.

For 2-channel data (Genepix) the value compared to the reference range is the ratio of the foreground/background intensities.

To Filter Based on Expression Threshold

1           In the Filtering Panel, select the Expression Threshold from the selection list.

2           Enter the Rang Min to define the reference range.

3           Enter the Rang Max to define the reference range.

4           Select the Filter Values from the dropdown list.

5           Click on Filter.  The View Area is updated to reflect the filter applied (after the screen has been refreshed).

Note: Filtering is performed directly on the loaded dataset; a separate copy is not generated.
