Deviation-based Filter

Sets to "missing" all markers whose measurement deviation across microarrays is below a given value.

The parameters for this filter are:

  1. Deviation bound: A non-negative value D. For every marker, the filter will compute the deviation for the measurements of the marker across all microarrays in the set. If the deviation is less than D, then all the measurements for the marker will be set to "missing".
  2. Missing values: When computing the deviation for a marker, it is possible that some of the marker's measurements are missing. The "Missing values" drop-down allows users to designate how to handle such missing measurements when computing the marker's deviation. Available options are:
    • Marker average: replaces a missing value with the average measurement of its corresponding marker. The average is computed using only non-missing values.
    • Microarray average: replaces a missing value with the average measurement of the microarray where the missing value belongs. The average is computed using only the non-missing values in the microarray.
    • Ignore: a missing value is ignored (i.e., treated as non-existent) in the computation of the marker's deviation.

For 2-channel data (Genepix) the measurements used in deviation computation are the ratio of the foreground/background intensities.
