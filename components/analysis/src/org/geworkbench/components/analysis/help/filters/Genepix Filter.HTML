Genepix Flags Filter

This filter operates on Genepix data. It makes use of the “Flags” column; specifically, it allows users to designate as “missing” measurements whose Flags column assume specific values.

How to Filter by Genepix Flags

1. Load the Genepix file (.gpr) into a project.

2. Navigate to filter pane in the Analysis Area and select Genepix filter. A pick list is displayed, populated with all the values found in the Flag column of the selected dataset.

3. Select values from the pick list to filter from the dataset.

4. Click on Filter. The application will then proceed to identify all markers with Flags settings designated by the user in the pick list, and will set the value of the marker "missing".
