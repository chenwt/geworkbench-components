Microarray-based Mean and Median Centering Normalizer

This normalizer first calculates the mean (median) measurement for all measurements in a microarray. Then it subtracts the mean (median) measurement from every measurement in the microarray.

The parameters for this normalizer are:

  1. Averaging method: Designates if the mean or the median should be used for centering. Mean and median values are computed from the non-missing values within the microarray.
  2. Missing values: Designates how to pre-process any missing values in a microarray, prior to centering them. Available options are:
    • Min profile: first centers all non-missing measurements within the microarray where a missing value belongs. Then replaces the missing value with the minimum among the centered values.
    • Max profile: first centers all non-missing measurements within the microarray where a missing value belongs. Then replaces the missing value with the maximum among the centered values.
    • Zero: replaces a missing value with 0.
    • Ignore: leaves missing values unchanged.
