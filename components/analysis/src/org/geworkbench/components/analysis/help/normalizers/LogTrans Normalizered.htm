Log2 Transformation Normalizer

The log2 transformation is applied to all measurements in a microarray. The operation is applied only if all values in all microarrays are positive. Otherwise an error message appears indicating to the user that the normalization could not be performed.

 Apply a Log2 Normalizer

1           In the Normalization Panel, select the Log2 Normalizer.

2           Click Normalize.  The View Area is updated to reflect the filter applied.

Note: Normalizing is performed directly on the loaded dataset; a separate copy is not generated.
