Expression Threshold Filter

Sets to "missing" all marker measurement that are outside (or inside) a user defined range.

The parameters for this filter are:

  1. Range Min, Range Max: These are real numbers that define a reference range [min, max] against which marker measurements will be compared .
  2. Filter values: Designates how the marker measurements are compared to the reference range. Available options are:
    • Inside range: Sets to "missing" a measurement if that measurement is inside the reference range.
    • Outside range: Sets to "missing" a measurement if that measurement is outside the reference range.

For 2-channel data (Genepix) the value compared to the reference range is the ratio of the foreground/background intensities.
