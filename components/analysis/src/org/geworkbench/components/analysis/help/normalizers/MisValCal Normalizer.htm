Missing Value Computation

This normalizer replaces a missing measurement with a consensus value. This value is computed based upon the selection at the drop-down box titled "Averaging method". Available options are:

· Mean profile marker: replaces the missing measurement with the average value (across all microarrays) of its corresponding marker. The average is computed by taking into account only non-missing values.

· Mean microarray value: replaces the missing measurement with the average value of its corresponding microarray. The average is computed by taking into account only non-missing values.

Apply Missing Value Normalizer

1 In the Normalization Panel, select the Missing Value Computation from the selection list.

2 Click on the Averaging Method drop down list and select Mean profile marker or Mean microarray value.

3 Click on Normalize. The View Area is updated to reflect the results of the normalization operation (after the screen has been refreshed).

Note: Normalizing is performed directly on the loaded dataset; a separate copy is not generated.
