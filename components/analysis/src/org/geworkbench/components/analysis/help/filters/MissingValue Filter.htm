Missing Values Filter

This filter discards from a microarray set all markers that have "missing" measurements in more than X microarrays (X is a user specified value). Markers are removed from all the microarrays in the set.

In the example shown below, the filtering operation will remove a marker if the marker has a missing value in 1 or more microarrays ("Maximum number of missing arrays" designates the maximum number of arrays where we are willing to tolerate a missing value).
