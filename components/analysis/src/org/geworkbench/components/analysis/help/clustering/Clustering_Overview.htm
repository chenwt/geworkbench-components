Clustering Overview

The application provides clustering tools for analyzing the data of any dataset in the project tree. Similar to the case of filters and normalizers, clustering methods are pluggable. This makes it possible for users/coders to define their own analysis plugins and use them from within the application. The clustering plugins that are loaded at any point are made available from the Analysis Panel. Upon selecting a clustering algorithm, you will be prompted to provide values for all the pertinent parameters.

Default configuration of the application supports the following clustering methods:

  • Hierarchical clustering Clustering of markers and or microarrays into a hierarchical tree.

  • Self-Organizing Maps Clustering of marker profiles into the cells of user defined grids.

The implementation of these analyses is based on code from the Multi Experiment Viewer (MEV) platform, made available from The Institute for Genomic Research (TIGR).
