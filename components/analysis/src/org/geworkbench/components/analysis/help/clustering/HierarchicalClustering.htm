Hierarchical Clustering

Hierarchical clustering is a method to group markers together based on similarity on their expression profiles and to organize the grouped markers in a tree-like hierarchical structure to indicate the strength of their correlations. This help section describes applying Hierarchical Clustering algorithm and visualizing the results.

Apply a Hierarchical Clustering Algorithm

1 In the Analysis Panel, select Hierarchical Clustering from the selection list.

2 Populate the parameters used for this analysis method:

Clustering Method: This parameter is used to indicate the convention used for determining cluster-to-cluster distances when constructing the hierarchical tree. Available options are:

Single Linkage: The distances are measured between each member of one cluster each member of the other cluster. The minimum of these distances is considered the cluster-to-cluster distance.

Average Linkage: The average distance of each member of one cluster to each member of the other cluster is used as a measure of cluster-to-cluster distance.

Total Linkage: The distances are measured between each member of one cluster each member of the other cluster. The maximum of these distances is considered the cluster-to-cluster distance.

。 Clustering Dimension: These are used to indicate whether to cluster markers, microarrays, or both.

。 Distance Metric: Euclidean Distance, Pearson Correlation and Spearman Rank Correlation are provided as options.

3 User specifies where to run the analysis in the Service subtab. The Service subtab allows the user to select which compute server to use: Local or Grid. The local instance of the server runs on the user's machine; GRID allows the user to pick up one among the available caGrid-enabled versions of Hierarchical Clustering.

4 Click Analyze . The Visual Area is updated to reflect the Dendrogram.

。 Error Message: If missing values are included in the data, a pop up message will state Microarray set contains missing values . To remove the missing values, run the Missing Value Computation Normalization. This is described in detail in Normalizers Help section.

Dendrogram Visualization

Basic Usage

。Enable Zoom: When Enable Zoom is selected, areas of interest can be mouse selected in the dendrogram. The selected region is magnified in View Area. To return to the original image, unselect Enable Zoom.

。Gene Height: Modifies (increasing or decreasing) the display of Gene Height.

。Gene Width: Modifies (increasing or decreasing) the display of Gene Width.

。Intensity Slider: Adjusts the dendrogram display intensity.

。Right click > Add to Set: The genes in the selected region is added as a new Cluster Tree Marker Set.

。Right click > Image Snapshot: The Dendrogram is captured as JPEG image and added to the Project Tree.

。Tool Tip (light bulb): When the tool tip is enabled (bright yellow,) moving the mouse pointer over any spot of the Dendrogram displays the chip, marker and signal corresponding to that point.
