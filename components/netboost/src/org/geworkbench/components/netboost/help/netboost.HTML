NetBoost

Netboost an algorithm used to assess evolutionary mechanisms that may have given rise to an interaction network. For precise algorithm details, see Chris Wiggins’ paper (http://arxiv.org/abs/q-bio.QM/0408010

Run NetBoost

1. In the Analysis tools section of the user interface, select NetBoost.

2. In the analysis pane (lower right), select Aracne analysis from the analysis list.

3. Populate the parameters used for this analysis method.

4. Select the location to run the analysis (local or Grid) in the service subtab.

5. Click on Analyze. If successful, the project window is updated to reflect the result node (a child node of the input dataset The Dataset history captures the analysis parameters.

Parameters

Main tab parameters are as follows:

· Training Examples: The number of training examples to generate for each candidate model M.

· Boosting Iterations: The number of boosting iterations in each training round.

· Subgraph Counting Method: Specifies how the feature space is defined, namely using subgraphs that can be constructed by any 8-step walk or using subgraphs that contain 7 edges.

· Cross-validation Folds: The number of cross validation folds to use.

Secondary tab parameters are as follows:

This tab lists the generative models to evaluate. Every model name is pre-pended with a check box allowing the user to specify which models to run NetBoost against.

View NetBoost Results

· You can click Image Snapshot button to captures the graphs as an image.

· Export: Click the Export button or, right click on the result node and select Export or, from the file menu select File>Export. The export file contains the Score Table and Confusion Matrix in a CSV File Format. Each of the fields in the table is separated with commas
