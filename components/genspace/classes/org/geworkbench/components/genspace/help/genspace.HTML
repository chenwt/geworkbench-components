genSpace Overview

genSpace is a tool that allows geWorkbench users to collaborate and share knowledge using a social networking metaphor. It transparently logs information about the analysis tools that geWorkbench users run, so as to answer such questions as "what are the most commonly used analysis tools?" or "which analysis tools are most commonly used together?" These questions are answered by graphical visualizations and by real-time suggestions that are integrated into the geWorkbench environment.

genSpace also incorporates an online wiki that provides registered users with the ability to comment on and rate different analysis tools and series of analysis tools, called "workflows".

Features
The online help includes further information about other genSpace features, including how to:

  • log in to genSpace and change your privacy settings
  • see visualizations of common workflows
  • get real-time suggestions about your current workflow
  • see statistics about all users' workflow and analysis tool usage
