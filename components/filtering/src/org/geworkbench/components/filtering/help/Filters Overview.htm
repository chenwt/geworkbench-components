Filters

The Filters pane is located in the Command and Analysis Area (see image below) in the lower right of the application. The application offers a selection of pluggable filters to assist in the manipulation of the data and in their preparation for analysis.

Filters are used in order to remove data points from an expression dataset, e.g. because some quality data criterion is not met. The result of applying a filter is (1) either to set the status of a value to "missing" or, (2) to completely eliminate a marker from the dataset. In the later case, all measurements for that marker (across all microarrays in the microarray set being filtered) will be eliminated.

Changes resulting from the application of a filter are performed directly on the selected dataset (i.e., a separate copy of the dataset is not generated). Available filters are displayed in the Filtering Panel (figure below). Experienced users can add or remove filters by manipulating the relative entries in the application configuration file (see the design documentation). Most filters have associated parameters that control their operation. Values for these must be provided in the appropriate areas within the Filtering Panel.

A variety of filters are available with the application:

Filters

o           Missing values filter: Discards all markers that have "missing" measurements in more than X microarrays, where X is a user specified value.

o           Deviation-based filter: Sets to "missing" all markers whose measurement deviation across microarrays is below a given value.

o           Expression threshold filter: Sets to "missing" all markers whose measurement levels are outside (or inside) a user defined range.

o           Affymetrix detection call filter: Applicable to data from Affymetrix arrays. Sets to “missing” all measurements whose detection status is A, P or M. 

o           2 channel threshold filter. Applicable to 2-channel arrays (Genepix). Defines applicable ranges for each individual channel. Sets to "missing" all values for which either channel intensity does not respect its defined range.

o           Genepix filter: Applicable to Genepix data. Uses the values in the “Flags” column to decide which marker values should be designated as missing.
