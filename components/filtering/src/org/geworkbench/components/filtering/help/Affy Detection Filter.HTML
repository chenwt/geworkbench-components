Affy Detection Call Filter

This filter operates on Affymetrix data. It sets to “missing” measurements whose detection call is A (absent), P (present) or M (marginal). The user designates which detection calls they want filtered.

To Filter Measurements based on Affy Detection Status

1 In the Filtering Panel, select the Affy Detection Call Filter from the selection list.

2 Select one or more detection calls to use as conditions for the filtered

3 Click on Filter. The View Area is updated to reflect the filter applied.

Note: Filtering is performed directly on the loaded dataset; a separate copy is not generated
