2-Channel Threshold Filter

This filter operates on Genepix data. It allows users to filter each channel signal separately. Similar to the expression threshold filter, users define reference ranges. Here, though, there are 2 reference ranges designated: one for the Cy3 channel (green) and one for the Cy5 channel (red). Marker measurement are set to "missing" if the intensity of either channel is outside (or inside) their corresponding range.

The parameters for this filter are:

Cy3 Range Min, Cy3 Range Max: Real numbers that define a reference range [min, max] against which the green channel intensities will be compared.

Cy5 Range Min, Cy5 Range Max: Real numbers that define a reference range [min, max] against which the red channel intensities will be compared.

Filter values: Designates how the channel intensities are compared to the reference range. Available options are:

o             Inside range: Sets to "missing" the measurement for a marker if either channel intensity is inside the corresponding the reference range.

o             Outside range: Sets to "missing" the measurement for a marker if either channel intensity is outside the corresponding the reference range.

The intensity for a channel is calculated by subtracting the background measurement from the foreground measurement for the channel.

To Filter Based on 2 Channel Threshold

1           In the Filtering Panel, select 2 Channel Threshold Filter from the filter selection list.

2           Enter the CY3 Rang Min to define the low end of the reference range for the green channel intensities.

3           Enter the CY3 Rang Max to define the high end of the reference range for the green channel intensities.

4           Enter the CY5 Rang Min to define the low end of the reference range for the red channel intensities.

5           Enter the CY5 Rang Max to define the high end of the reference range for the red channel intensities.

6           Select the Filter Values from the dropdown list.

7           Click on Filter.  The View Area is updated to reflect the filter applied.

Note: Filtering is performed directly on the loaded dataset; a separate copy is not generated.
