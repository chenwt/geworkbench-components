ARACNE

ARACNE (Algorithm for the Reconstruction of Accurate Cellular Networks) is an information-theoretic algorithm used to identify transcriptional interactions between gene products using microarray expression profile data and visualize the analysis results. ARACNE predicts potential functional associations among genes, or novel functions for uncharacterized genes, by identifying statistical dependencies between gene. This help topic describes executing ARACNE and viewing the results a matrix of candidate interactions, also called an adjacency matrix, which can be used for further network visualization and analysis.

Run ARACNE

1. A microarray set or adjacency matrix selected in the project folders area of geWorkbench.

2. In the analysis pane (lower right), select ARACNE analysis from the analysis list.

3. Populate the parameters used for this analysis method.

4. Select the location to run the analysis (local or Grid) in the service subtab.

5. Click on Analyze. If successful, the project window is updated to reflect the result node (a child node of the input dataset). The Dataset history captures the analysis parameters.

Parameters

· Hub Marker(s): This drop-down specifies the Hub Markers.

· All vs. All: MI values are computed for all pairs of eligible markers.

· List: Supports manually entering a list of comma separated marker ids (or loading the list from a CSV file by clicking Load).

· Threshold Type: This drop-down specifies the type of threshold to be used and assumes the values “Mutual Info” or “P-value”. This value is always a number between 0 and 1.

· Kernel width: The kernel width used for the computation of the MI values. This can be either inferred by the application (select Inferred) or entered explicitly by the user (select Specify).

· DPI Tolerance: If the “Do Not Apply” option is specified, no DPI is applied. If the “Apply” option is selected then the DPI is applied and the value (between 0 and 1) in the associated text box is used to determine the stringency level of the DPI application.

· DPI Target List:

· All Markers: ARACNE will compute MI values between pairs of eligible markers. The “All Markers” checkbox determines which markers qualify as eligible. Specifically, if the “All Markers” checkbox is not selected then only activated markers are eligible to participate in MI computations; otherwise, all markers in the microarray set are eligible.

· All Arrays: If the “All Arrays” is not selected then only values from activated microarrays participate in the computation of the MI values. Otherwise values from all microarrays are used.

View Arance

Execution of ARACNE generates an adjacency matrix which can be visualized and further explored in Cytoscape. A description of Cystoscape is located in the Cytoscape Tutorial and Help.
