Alignment Results

The Alignment Result component is located in the View Area in the upper right of the application. This visualization component displays Blast results from the Analysis Area Sequence Alignment component. When Blast results are returned, they are placed in the Project Folders as a child node of the sequence they correspond to. Scroll over the data node created and the tooltip will indicate the number of sequences in the file.

The results of a Blast search is displayed both as Paracel BLAST in a separate window and BLAST alignments in the Alignment Result component. This help section describes Alignment Results and how to create a fasta file with sequences selected.

The alignment results are divided into the following five areas:

1. The list of markers included in the Blast results. Select a marker from this panel to the right (2) is updated. The Search box and Find Next are used to locate markers within this list.

2. This lists the Blast hits for the selected maker. This information is also displayed in a separate Web Browser (see Blast Advanced Options).

3. The selected sequence details are displayed in this section of the alignment results. Select sequences of interest to add back to the main project by checking the include checkbox or Select All button. This includes hyperlinks to NCBI public databases sequence description.

4. This section includes the following buttons:

Load: Loads stored html alignment file.

Reset: Clears the selected sequences.

Selects all: This is used to select all checkboxes to include all of the sequences for the selected Marke.

Add Selected Sequences to Project: A FASTA file is created with the included sequences selected after this button is selected. Scroll over the data node created and the tooltip will indicate the number of sequences in the file.

Only Added Alignment Parts. Select sequences and the click on Only Added Alignment Parts to create a FASTA file that contains portions of the sequence that aligns with the input data. Scroll over the data node created and the tooltip will indicate the number of sequences in the file.

5. Displays the total number of hits for the selected sequence.

Adding Sequences to a Project

1 Click on the Include checkboxes to selected sequences of those hits. Click on Select All to select all of hits.

2 Click on Add Selected to a Project. The program will generate a temporary FASTA file for those sequences; a new file node will be added under current project.

3 (If desired) Right click on the file node to save or rename.
