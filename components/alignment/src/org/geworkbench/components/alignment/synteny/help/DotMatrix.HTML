Dot Matrix Panel

The Dot Matrix is located in the View Area in the upper right of the application. This is a visualization comparison of two genomic regions with detailed annotation for both regions.

Basic Usage

Tool Bar: This is a series of radio buttons that modify the display of the Dot Matrix.

– Direct On/Off and Invert On/Off - Hides or displays results of direct to direct or direct to complementary comparisons.

– Annotation X On/Off and Annotation Y On/Off - Hides or displays annotation for sequences along X or Y axis.

– Save Image - Stores the current picture in jpeg format.

Info Bar: Located below the Tool Bar, this section of the Dot Matrix provides chromosome mouse over information.

–The top row in the Info Bar section provides current mouse position information. If the mouse is in the Dot Matrix Area absolute and relative sequence position will be displayed. If the mouse pointed at Annotation Feature the name, position and Annotation Track name will be displayed.

– The bottom row in the Info Bar section provides the title of the currently displayed comparison including chromosome number and segment positions.

Annotation Area

– Annotation Area consists of separate Annotation Tracks and each track consists of Annotation Features. Most of displayed data obtained from UCSC server (golden Path, http://genome.ucsc.edu ) some other tracks were computed by server side part of Synteny Component. Currently only PFP track (Paracel Filtering Package, Paracel inc.) for identifying repeats and low complexity regions is computed by server side application. Use Annotation subtab in the Synteny tab in Analysis area to change hide/display annotation tracks.

– Mouse over an Annotation Feature to display its name, position and Annotation Track name. If the feature has a related URL, the user can click on the feature to view that url in a new browser.
