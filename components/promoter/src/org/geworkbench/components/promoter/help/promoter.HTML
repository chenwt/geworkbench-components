Promoter Panel

The Promoter Panel is located in the View Area in the upper right of the application. This visualization panel uses the JASPAR Transcription Factor Binding Profile Database (http://jaspar.cgb.ki.se/cgi-bin/jaspar_db.pl ). The promoter panel can used to visualize transcription factor binding sites (TFBS) using the following:

Transcription Factor signature analysis

n Searching putative regulatory sequences against a collection of known transcription factor DNA-binding signatures represented as a position weight matrices (PWMs) [Lawrence and Reilly, 1990].

This help section describes promoter panel basic usage, viewing transcription factor analysis and view results from Pattern Discovery.

Basic Usage

The steps of basic usage are indicated with letter in the above screenshot.

A. TF Mapping or Module Discovery: This tab allows users to select A.TF Mapping or Module Discovery: This tab allows users to select TF Mapping or Module Discovery. The Module Discovery Tab functionality is not yet implemented.

B. TF list: The TF list contains the names of available transcription factors signatures from Jaspar or those that have been loaded from files. This lists of known Transcription Factor (TF) binding site sequence patterns. Each is represented by a Position Specific Scoring Matrix (PSSM).

C. Double click to move TF signatures from this list box to the Selected TF (lower box). This list box shows transcription factors which can be searched against the available genomic sequences by clicking on Scan. Double-clicking on a TF name clears it from the Selected TF list.

D. Scan: Scan the sequences in the Selected TF against the available genomic sequences.

E. Retrieve: Loads file of stored promoter sequences.

F. Load: Loads a file containing new transcription factors to add to the TF Listing.

G. Save: Stores the sequence data in the display area.

H. The lines represent the full 4000 base pairs of each sequence. Double click on the sequences to see the textual representation of the DNA and the TF signature match, shown as a colored rectangle.

I. The bottom of the window includes a sequence logo representation of the TF signature. Pattern and the relative weights of each component are displayed at bottom in a logo-style graphic.

J. Stop: Stops the scanning process.

K. The progress bar used to provide a visual update of the mapping percentage completion.

L. Clear: Clears the display area of active patterns and active TFs. This is useful mostly for resetting the "active TFs" data structure which is not automatically cleared before each Scan operation.

M. Show patterns : Controls if the patterns currently in the "active patterns" structure are displayed (as ovals) over the sequences. When selected (checked), the patterns are displayed. Unselected, the patterns are not displayed.

N. Show TF: Controls if the TFs currently in the "active TF" structure are displayed (as rectangles) over the sequences. When selected (checked), the display area will include TF’s in the sequence display. Unselected, the TF’s are not displayed.

View Transcription Factor Analysis

1 Load a FASTA File into the project pane.

2 Select promoter panel. The display area is updated to reflect the sequences for the FASTA file loaded.

3 Click on TF signatures to move from the TF list to the Selected TF list.

4 Click on Scan, the e Selected TF s are searched against the available genomic sequences. The patterns will be mapped onto the sequences according to match criteria below the Save and Retrieve buttons. If matches are found, they are returned in various colors. Individual hits can be identified by positioning the mouse pointer over them, which will display the pattern name.

。 Note: Each time that a transcription factor (TF) matching operation is run, the "active TFs" data structure is *AUGMENTED* with the results of the discovery operation (i.e., contents due to previous runs are maintained). The "active TFs" is not affected by pattern discovery.

View Pattern Discovery Results

1 Load a FASTA File into the project pane.

2 Run Pattern Discovery analysis. (See Pattern Discovery online help for more information)

3 In the Pattern Discovery Panel discovery table, left click on the row(s) to display selected motifs and their frequencies in the dataset. The Promoter Panel, Sequence Panel, and Patterns are updated to display the patterns selected.

。 Note: Each time that a pattern discovery is run, the contents of the "active patterns" structure are REPLACED with the results of the discovery operation (i.e., contents due to previous runs are cleared).
