Marker Panel

The marker panel shares the bottom-left pane of the application with the Phenotype panel. Click the "Marker Panel" header on the pane to see its content. A workspace can have multiple marker panel sets, each includes multiple marker panels. Only one marker panel set is active at any given time. Scroll down the selection box to choose a marker panel set, the selected marker panel set and marker panels in that set are displayed below in a tree structure. Click the "+" sign in front of a marker panel name to view the markers in that panel.

The following operations can be applied to marker panels:

Create Panel

Create Panel Set

Save/Load Panel Set

Save/Load Panel

Rename Panel

Delete Panel

Activate/Deactivate Panel

Search Marker Name Name in Panel
