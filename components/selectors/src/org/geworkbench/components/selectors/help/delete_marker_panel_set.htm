Savinga Panel

The save function is used to externally store Panels. Perform the following steps to Save a Panel:

1. Right Click on the panel name and select Save to bring up the Save Marker Panel dialog box.

2. The drop-down menu with the default of caWorkbench is used to select the location to save the panel.

3. Then enter the panel name in the File Name text box. A drop-down menu offers two File Types: Marker Panel Files (default) or All Files.

4. Select a file type and then press the Save button on the dialog box.
