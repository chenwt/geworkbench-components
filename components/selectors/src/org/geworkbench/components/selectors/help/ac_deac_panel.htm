Activate a Panel Set

Activated panel sets affect the contents displayed in the various visualization components. Most such components provide an option for imposing a panel-view. When such a view takes effect, the component will only display a subset of the original data that contains those markers (microarrays) that are contained in activated marker panels (phenotypes). An activated panel is available to other caWorkbench components. Unless it is activated, a panel will not be taken into consideration by other components. More than one panel can be active at the same time. Perform the following steps to activate a panel:

- Click on the panel and then perform one of the following actions:

。 Right-click on the panel and select Activate

。 Select the checkbox next to the panel name.

。 Through the main menu> Commands Panel > Activate Panel.
