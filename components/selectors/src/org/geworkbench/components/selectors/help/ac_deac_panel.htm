Activate/Deactivate Marker Panel

To activate a marker panel, select the marker panel and right-click the mouse. Choose the "Activate" option to activate that marker panel. After activated, the name of the panel will appear highlighted.

To deactivate a marker panel, select the marker panel and right-click the mouse. Choose the "Deactivate" option to deactivate that marker panel.

Activated panels affect the contents displayed in the various visualization components. Most such components provide an option for imposing a "panel-view". When such a view takes effect, the component will only display a subset of the original data that contains those markers (microarrays) that are contained in activated marker panels (phenotypes).
