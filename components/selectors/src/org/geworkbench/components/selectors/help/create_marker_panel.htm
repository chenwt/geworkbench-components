Create a single panel

Choose a marker panel set in which a new marker panel will be created by selecting a marker panel set from a pull-down list in the bottom-left pane of the interface ("Marker Panels" pane). Move the mouse over the maker panel set name and right-click the mouse. Choose the "New Panel" option, which will create a marker panel with a default name "New Panel" and display the newly created marker panel as a leaf under the marker panel set tree.
