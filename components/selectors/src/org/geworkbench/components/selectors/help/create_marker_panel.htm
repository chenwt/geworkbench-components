Deactivating a Panel Set

Deactivation is used to disable an activated panel set. The deactivated panel set will not be available to other components. An activated panel cannot be deactivated other than by applying the Deactivate function. Perform the following steps to deactivate a panel:

- Click on the panel and the selected panel label will be highlighted. Then perform one of the following actions:

。 Right-click on the panel name and select Deactivate.

。 Unselect the checkbox next to the panel.

。 Through the main menu select Commands Panel > Deactivate Panel.
