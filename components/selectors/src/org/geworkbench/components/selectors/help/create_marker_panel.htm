Deactivate

Deactivate is used to disable the activation of an already activated set. The deactivated set will not be available to other components. An activated set cannot be deactivated other than by applying the Deactivate function. Perform the following steps to deactivate a set:

Click on the set and the selected set label will be highlighted. Then perform one of the following actions:

。 Right-clicking on the set and select Deactivate.

。 Unselecting the checkbox next to the set.

。 Through the main menu select Commands Panel > Deactivate Panel.
