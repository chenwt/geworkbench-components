Panels overview

Panels are collections of either gene markers (gene marker panel) or microarrays (phenotype). Panels provide a convenient mechanism for grouping together markers and microarrays of interest. They are used to dissect the potentially massive expression data into more manageable chunks.

Gene marker panels and phenotypes are displayed separately (in the "Marker Panel" and "Phenotype Panel" areas respectively). Panels can be grouped into panel sets based on shared features. Panel set names, panel names and their components are displayed in a tree structure.

The following operations are supported for panels:

  • Create panel set & panel
  • Save/Load panel set
  • Rename panel set and panel
  • Delete panel set and panel
  • Activate/Deactivate panel
  • Search for a named panel
  • Create DEFAULT panels in Marker Panel component
