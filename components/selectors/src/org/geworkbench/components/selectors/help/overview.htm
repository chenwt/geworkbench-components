Overview

The Phenotype and Marker tabs are located in the Selection Area in the lower left of the application. These are collections of either gene markers (marker tab) or microarrays (phenotype/array tab). These provide a convenient mechanism for grouping together markers and microarrays of interest according to the user’s preferences. They are used to dissect the potentially massive expression data into more manageable chunks.

Gene marker and phenotypes are displayed separately (in the "Marker" and "Phenotype/Array" tabs respectively). Markers and Phenotypes can be grouped into sets based on shared features.

The following operations are supported:

  • Create group

  • Save a set

  • Rename set

  • Delete a set

  • Activate a set

  • Deactivate a set

  • Load a set
