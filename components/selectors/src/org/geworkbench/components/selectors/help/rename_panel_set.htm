Renaming a panel set

To rename a marker panel set onto disk, select that marker panel set in the "Marker Panels" pane and right-click the mouse. Choose the "Rename Panel Set" option and provide new marker panel set name in the pop-up window and click "OK" to rename the set.

Another way to rename a marker panel set is to first select the marker panel set to be renamed and choose the "Rename" option on the "file" pull-down menu. After choosing the "Panel Set" option and the "Marker" option, provide new marker panel set name in the pop-up window and click "OK" to rename the set.
