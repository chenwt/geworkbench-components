Renaming a Panel Set

The Rename function is used to change the name of a Panel label. Perform the following steps to rename a panel:

1. Click on the panel and the selected panel label will be highlighted. Then perform one of the following actions:

。 Right-click on the panel name and select Rename

。 From the main menu select Commands Panel > Rename

2. Then enter the panel name in the File Name text box and click Ok.
