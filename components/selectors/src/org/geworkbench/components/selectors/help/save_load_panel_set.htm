Saving/loading a panel set

To save a marker panel set onto disk, select that marker panel set in the "Marker Panels" pane. Choose the "Save" option on the "File" pull-down menu and choose the "Panel Set" option and then the "Marker" option. In the "Save Marker Panel Set" pop-up window, provide the directory and file name to be saved and click "Save".

To load a marker panel set, choose the "Open" option on the "File" pull-down menu and choose the "Panel Set" option and then the "Marker" option. In the "Open Marker Panel Set" pop-up window, provide the directory and file name to be opened and click "Open". The newly opened marker panel set becomes the active marker panel set in the application.

Another way to load marker panel set is to move the mouse over an existing marker panel set name in the "Marker Panel" pane and right-click the mouse. Choose the "Load Panel Set" option, which will bring up the "Open Marker Panel Set" window. Follow the instruction above to load a new marker panel set.
