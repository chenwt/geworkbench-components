Load a Panel

The load function allows you to load a previously saved panel directly into the Gene Panels frame. Perform the following step to retrieve a saved panel:

1 Select Load Panel, this brings up the Open Marker Panel dialog box.

2 Scroll to the file location using the Look In: drop-down menu.

3 Select the panel of interest and then click on Open. The saved panel will appear at the bottom of the list of panels in the lower frame.
