T-Test Analysis

The t-test determines if for each marker there is a significant difference between two groups, such as case or control classification described in the Marker and Phenotype section of help. This statistic accomplishes this task by examining the differences between the means relative to the variance of the data. The formula for the t-test consists of determining the ratio of the differences between the two means and the measure of the variability between two datasets or panels.

The T-test analysis implementation is based on code from the Multi Experiment Viewer (MEV) platform, made available from The Institute for Genomic Research (TIGR). The MeV Manual offers a detailed description of the workings of MeV. T-Test results are visualized in the Volcano Plot and Color Mosaic.

Apply T-Test

1 In the Analysis tools section of the user interface, select T-Test Analysis .

2 Update or accept the default setting for the parameters as described in the parameter section.

3 Click on Analyze. In the Marker tab, the markers which met the significance test are included in a new set called “Significant Genes”. The values of the T-Test can be seen in the Parameter Settings

    • P-Value

    This tab indicates the method by which p-values are determined for each gene and allows the input of the critical p-value. The steps of basic usage are described as follows:

    - p-values based on t-distribution: Using this option, a gene's p-value is taken directly from the theoretical t-distribution based on the gene's calculated t-value.

    - p-values based on permutation: A gene's p-value is determined by forming a distribution based on permutations of the data for that gene.

    - Randomly grouped experiments: This text box captures number of times to permute the values.

    - Use all permutations: Permutes the values a number of times equal to the maximum number of permutations possible.

    - Overall alpha: A text box to capture the alpha or critical p-value.

    - Activated Markers and Activated Array Checkbox: These checkboxes limit the analysis to activated sets.

    Degree of Freedom

    The degrees of freedom for an estimate is defined as the number of independent measurements that go into determining an estimate. It is directly related to the number of data points and is dictated by the microarray experimental design.

    When assuming Equal Group Variances, the degrees of freedom are the sum of the number of samples in both categories (panels) minus 2 (As the variance and mean for the groups are calculated during the T-test classification process).

    For unequal group variances, the degrees of freedom are computed using the Welch’s t-test approximation which produces a slightly smaller t-value than the traditional equal variance T-test (also called Student’s T-test). The computed t-value is used above to determine the significance p-value for each gene using parameters based on above text-box. The steps of basic usage are described as follows:

    Group Variance:

    - Equal group variances: Unpaired data with equal variance.

    - Unequal group variances: The approximate degrees of freedom are obtained from Welch’s approximation formula.

    Alpha Corrections

    This is an estimate of the risk level for the estimate of significance, or a rate for acceptable false error. Traditionally this is set as 0.05 (called Type I error), or that there is 1 in 20 chance that the T-test classification of genes as significant in either category would be incorrect.The steps of basic usage are described as follows:

    Correction Method:

    -Just Alpha (no correction): Alpha is not altered.

    -Standard Bonferroni Correction: A user-specified alpha is divided by the number of genes to give the critical p-value.

    -Adjusted Bonferroni Correction : T-values for all the genes are ranked in descending order and the gene with the highest t-value, the critical p-value becomes (alpha / n), where n is the total number of genes; for the gene with the second-highest t-value, the critical p-value will be (alpha/ n-1).

    -Step-down Westfall Young: These functions compute permutation adjusted p-values for step-down multiple testing procedures described in Westfall & Young (1993).
