Storing Parameters for Clustering Algorithms

FIlters, Normalizers, and clustering methods such as Hierarchical clustering all have one or more parameters associated. Storing the parameters for a specific use of a method allows the uniform application of the method on several datasets.

To save parameters for a analytical method, provide a value for each parameter for the selected analytical method and click the "save parameter" button. In the pop-up window, provide the directory and the name under which the parameter settings for the selected method will be saved and click "Save". The name of the saved parameter set will be displayed in the parameter window for the selected method.
