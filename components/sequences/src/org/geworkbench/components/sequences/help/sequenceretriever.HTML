Sequence Retriever

The Sequence Retriever is located in the top-right of the application. This component retrieves sequences for the active markers listed in the Selected Markers list. Sequences found can be added to project as a sequence dataset and viewed in the Promoter Panel, Sequence Panel and Position Histogram.

Basic Usage

Selected Markers: From this list of markers in active marker panels, the application will obtain sequences in which to search for putative promoters

Type: Dropdown selection box to indicate DNA or Protein.

Region: Only valid for retrieving DNA sequence, these text boxes define the length of retrieved sequence. For example, -800 to +800 define regions of 1600bp DNA sequences which starts from upstream 800 bp location.

Get Sequence: Click onGet Sequenceto retrieve sequences. If no sequences are found for the selected markers, a pop-up message indicates that none are found. If no sequences are found, the sequence overview is displayed as lines. Use mouse over to view the location of current spot in the whole retrieved sequence. Sequences can be viewed by double clicking a line and to return to the sequence overview, double click again.

Add to Project: Click Add to Project to save the sequences as a sequence dataset in the project.
