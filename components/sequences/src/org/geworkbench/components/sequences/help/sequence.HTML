Sequence Panel

The Sequence Panel is located in the top-right of the application in the Visualization area. In this view, each sequence in the selected dataset FASTA file dataset is displayed as a thin line proportional to the sequence length in the Line View and as character strings in the Full Sequence View. This help topic describes basic navigation and viewing sequences with patterns.

Navigating a Sequence Panel

The steps of basic usage are indicated in the above screenshot.

1. View: Select the display format of the sequence data. In the Line View, each sequence is represented by a line proportional to its length, preceded by the sequence ID. The Full Sequence view displays character strings preceded by the sequence ID.

2. All / Matching Pattern: Limits the display to sequence with Patterns.

3. All Sequence: Limits the display to sequences in an activated Marker group.

4. Mouse over a sequence and the numeric position is displayed in a tooltip. Click on the individual lines to display the character string in the sequence details and at the bottom of the Sequence Component.

5. Sequence Display: A single click on a line or sequence string, the Sequence Display window displays character strings of the section of the sequence string selected. The display of sequence details is proportional to the click position in the line. If you click at the beginning of the line, the bottom will show the first part of the sequence. If you click the center place of a line, the Sequence display will show the middle part of whole sequence.

6. Arrows: Allows for scrolling to view the right/left of character string in the Sequence Display window.

Viewing Sequences with Patterns

Use the Pattern Discovery Module to search for Patterns in the Selection Area in the lower-left of the application. The patterns found can be viewed by selecting the hit(s) of interest. Each pattern (row) selected in the Pattern Discovery Module is displayed as a unique color block in the Sequence Component View. Click on a pattern block (in the Line or Full Sequence view) and the location of the pattern is displayed in the Sequence Display.

The tooltip format: numeric position, pattern (numeric position of the first character of the pattern, numeric position of last matching pattern character).

For example: 99, KG.GASGSF (97,105)
