Marker-based Centering Normalizer

This normalizer first calculates the mean (or median) measurement from all measurements in a marker profile. Then it subtracts this value from every measurement in the profile.

The parameters for this normalizer are:

Averaging method: Designates if the mean or the median should be used for centering. Mean and median values are computed from the non-missing values within the profile.

Missing values: Designates how to handle missing values in a marker profile. Available options are:

o             Min profile: missing values are replaced with the minimum (post-centering) profile value.

o             Max profile: missing values are replaced with the maximum (post-centering) profile value.

o             Zero: missing values are set to 0.

o             Ignore: missing values are left untouched.

Apply a Marker-based Normalizer:

1           In the Normalization Panel, select the Marker-based centering.

2           Select the Averaging Method from the dropdown list.

3           Select how to process the missing values from the Missing Value dropdown.

4           Click on Normalize.  The View Area is updated to reflect the normalization applied.

Note: Normalizing is performed directly on the loaded dataset; a separate copy is not generated.
