Normalizers

The application offers a selection of pluggable normalizers to assist in the manipulation of the data. The Normalizer panel is located in the Commands/ Analysis area in the lower right side of the application.

Normalizers are used to perform some needed data transformation, usually in preparation of an analysis step to follow. Normalizing a dataset will not give rise to a new dataset. Normalization results are applied “in place”, substituting pre- with post-normalization values. 

Apply a Normalizer

1           In the Normalization Panel, select the appropriate normalizer from the selection list.

2           Enter associated parameters that control the normalization operation.

3           Click on Normalize.  The View Area is updated to reflect the filter applied (after the screen has been refreshed).

Note: Normalization is performed directly on the loaded dataset; a separate copy is not generated.

Save Normalization Settings

4           Click on Save Settings which will result in a pop up window “New Parameter Setting Name”.

5           Enter a text description and click on Ok.

6           The description will appear in the text box to the right of the Normalizer selection.  The Normalization parameters are now saved along with other user settings. Saving these settings does not store a separate copy of the dataset.

Some of the normalizers available are the following:

n            Missing value computation: Replaces every "missing" value with either the mean value of its marker across all microarrays or with the mean measurement of all markers in the microarray where the missing value belongs.

n            Log transformation: The log2 transformation is applied to all measurements in a microarray. The normalization will not be performed (and an error message will be displayed) if the data set being normalized contains non-positive values.

n            Threshold value: All data points whose value is less (or more) than a user-specified minimum (maximum) value are raised (reduced) to that minimum (maximum) value.

n            Marker-based centering: Subtracts the mean (or median) measurement of a marker profile from every measurement in the profile.

n            Microarray-based centering: Subtracts the mean (or median) measurement of a microarray from every measurement in that microarray.

n            Mean-variance normalizer: For every marker profile, the mean measurement of the entire profile is subtracted from each measurement in the profile and the resulting value is divided by the profile’s standard deviation.

n            Quantile normalizer: Adjusts expression levels so that the expression value distribution of each microarray is identical.

Experienced users can add or remove normalizers by manipulating the relative entries in the application configuration file (see the design documentation).
