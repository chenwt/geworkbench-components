Threshold Normalizer

In this normalizer, all measurements are adjusted to a user specified maximum or minimum value. The parameters for this normalizer are:

Cut-off value: The user defined threshold value.

Cut-off type: The cut-off type drop-down designates the threshold requirement. Available options are:

o             Minimum: This selection will force all measurements below the threshold to be replaced by the cut-off value.

o             Maximum: This selection will force all measurements above threshold to be replaced by the cut-off value.

Missing values: The Missing values drop-down designates how to handle missing measurements. Options are:

o             Ignore: Leaves missing values unchanged.

o             Replace: Replaces each missing value with the threshold.

Apply Threshold Normalizer

1           In the Normalization Panel, select the Threshold Normalizer.

2           Enter the Threshold Cut-off Value in the text box.

3           Select the Cut-off type from drop down list and select “Minimum” or “Maximum”.

4           Select the Missing Values from drop down list and select “Ignore” or “Replace”.

5           Click Normalize.  The View Area is updated to reflect the normalization results.

Note: Normalizing is performed directly on the loaded dataset; a separate copy is not generated.
