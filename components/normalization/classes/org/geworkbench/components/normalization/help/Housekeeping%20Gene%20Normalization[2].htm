Housekeeping Gene Normalization

This Housekeeping Gene normalization method permits the normalization of data using the expression values of housekeeping genes.  In particular, for every microarray the average expression of all markers corresponding to housekeeping genes is calculated and then all expression levels within the microarray are divided by that average. To specify the housekeeping gene markers, the user can load a CSV (comma separated values) file containing these markers.  The loaded markers are displayed in the Current Selected Genes list box in the Normalization panel.  The list can be modified in the normalization panel and the resulting marker set can be saved as a CSV file and stored externally for reuse. 

 

This help section describes: basic usage, applying the normalizer, saving the selected genes to external location and when selected genes are not included..

 

Basic Usage

The steps of basic usage are indicated with letters in the above screenshot.

A.      Lists the genes removed from the Current Selected Genes list box (by double clicking or using ‘<’ icon).  The number in the brackets indicates the number of genes in the list box.

B.      Lists the markers that will be used for the normalization. The number in the brackets indicates the number of genes in the list box.

C.      Moves selected Excluded Gene(s) to the Current Selected Genes and updates the Number Current Selected Genes/ Excluded Gene “[#]”.

D.      Moves selected Excluded Gene(s) to the Current Selected Genes and updates the Number Current Selected Genes/ Excluded Gene “[#]”.

E.      Removes all markers from both list boxes.

F.      Prompts the user to designate a CSV file containing a list of housekeeping genes to be loaded into the Current Selected Genes list box.

G.      Saves the Current Selected Genes to an external location for reuse.

H.      Designates how to handle missing measurements in housekeeping genes. There are 2 options: either ignore missing values in the computation of the average or replace missing values with the average microarray value.

I.       (Right click Current Selected Genes) Modifies the name of the gene.

J.       (Right click Current Selected Genes) Removes the gene from the list and the updates the Number Current Selected Genes “[#]”.

K.      Stores the parameters of the normalizer which includes Missing Values Parameters, Current Selected Genes

L.       Applies the normalizer.

M.     A listing of previously saved parameter setting.

 

 Apply Housekeeping Gene Normalizer:

1           In the Normalization Panel, select the Housekeeping Gene Normalizer from the list.

2           Click Load and navigate to the CSV Housekeeping file.

3           Click on the file name and Open.  The system loads the file and populates the Current Selected Genes list box.

4           Manipulate the list of genes until the Current Selected Genes contains the full list of housekeeping genes to be applied in the normalization.

。              To remove marker(s) from the Current Selected Genes list box, double click on a marker, or select one or more markers and click on the left arrow “<”.  The marker(s) are removed from the Current Selected Genes list box and appear on the Excluded Housekeeping Genes list box.  Repeat this step until the Current Selected Genes contains the full list of housekeeping genes you wish to use.

。              To add genes back to the Current Selected Genes list box, double click on the gene, or click on the gene and click on the right arrow “>”.

。              To remove all values from both list boxes, click on Clear All.

5           Click on Normalize.

。              If all of the markers  listed in the Current Selected Genes list box are present in the microarray set being normalized, the View Area is updated to reflect normalization (after the screen has been refreshed).

。              If some markers are NOT present in the dataset, the system generates a warning message and highlights the missing values yellow in the list box.  See Genes not in the loaded dataset for additional description.

Note: Normalizing is performed directly on the loaded dataset; a separate copy is not generated.

 

When Selected Genes are not Included:

When you have attempted to normalize the data with genes that are not included in the data, the system generates a warning message and highlights the missing genes yellow in the Current Selected Genes list box:

。              If OK is selected – The system proceeds to normalize the data using only the markers that are present in the dataset and disregarding the highlighted markers.

。              If Cancel is selected – The system cancels the normalization operation.  The application indicates which genes are missing by highlighting the missing values.

Right Click Actions:

The application supports managing the highlighted genes by right clicking in the Current Selected Genes list box and selecting one of the actions described in the table below.

Action

Description

Exclude highlighted

Moves all the yellow highlighted markers into the Excluded Genes list box.

Delete highlighted

Deletes all the yellow highlighted markers from the Current Selected Genes list box.

Clear highlighted

Clears all yellow highlights.

 

Save a Housekeeping Gene List:

The actual list of housekeeping genes used to perform the normalization can be saved to an external location for reuse.

1           Click on the Save button. The system displays the Save Housekeeping Gene Panel pop-up window.

2           Choose the directory where to save the file.

3           In the File name box, enter the desired name.

4           Click Save.  The file is saved in CSV format.
