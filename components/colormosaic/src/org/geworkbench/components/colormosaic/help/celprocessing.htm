Color Mosaic

The Color Mosaic is located in the View Area ( see figure below) in the top-right of the application. This is a utility to view Eisen plot views of a Microarray dataset. Measurements in the selected microarray dataset are displayed as a two-dimensional matrix with microarrays as columns and genes/markers as rows. The color spectrum is used to indicate the relative magnitudes of the measurements. The relative colors are obtained for each probe (or spot) independent of the other probes in the dataset. Over- (and under-) expression is represented by relative values of expression of a probe across all micro-arrays in a dataset. Visualization preferences for the color mosaic are defined in the Preferences menu (Tools> Preference). These visualization preferences selection of Absolute or Relative impact the display as follows:

Relative:

。 Red represents over-expressed values. The brighter red (more intense) is used for larger positive values.

。 Blue represents under-expressed values. The brighter blue (more intense)is used for larger negative values

。 White indicates mean values of expression.

Absolute:

。 Green represents under-expressed ones. The brighter green (more intense) is used for larger negative values.

。 Red represents over-expressed values. The brighter red (more intense) is used for larger positive values.

。 Black values close to zero are displayed as variants of the black

Navigating the Color Mosaic

。 Printer Icon: Prints the displayed image.

。 Display: Must be toggled on to display data.

。 Accession: Displays marker accession numbers to the right of the mosaic.

。 Label: Displays marker annotations to the right of the mosaic.

。 'All Arrays' and 'All Markers': When not selected, subsets of the markers and microarrays which have been activated are displayed. The selections are created from the activated Marker and Phenotype Sets. When selected, all markers and/or all arrays will be displayed.

。 Intensity slider: Widget enables user to change the color intensity of the mosaic display. The color of the displayed markers changes linearly with slider movement.

。 Gene Height: Widget lets user change the height of each marker/gene row to a required number of pixels.

。 Gene Width: Widget lets user to change the width of each column in the mosaic to a required number of pixels.

。 Clicking on the mosaic: Clicking on any of the spots in the mosaic highlights the marker clicked in the marker set.

。 Image Snapshot: Right clicking on the mosaic will bring up the option to capture an image snapshot of the color mosaic.
