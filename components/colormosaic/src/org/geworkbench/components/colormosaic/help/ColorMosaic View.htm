Color Mosaic View

The Color Mosaic View is located in the View Area ( see figure below) in the top right of the application. This is a utility to view Eisen plot views of a Microarray dataset. Measurements in the selected microarray dataset are displayed as a two-dimensional matrix with microarrays as columns and genes/markers as rows. A blue-red color spectrum is used to indicate the relative magnitudes of the measurements. Red represents over-expressed values while blue represents under-expressed ones. The relative colors are obtained for each probe (or spot) independent of the other probes in the dataset. Over- (and under-) expression is represented by relative values of expression of a probe across all micro-arrays in a dataset. White indicates mean values of expression.

To view a dataset using Color-mosaic View

。 Select a dataset in the Project Panel and then select the Color Mosaic option in the View Area at the top-right section of the interface.

Navigating a Color-mosaic View

。 Clicking on mosaic: Clicking on any of the spots in the mosaic highlights the marker clicked in the Marker Panel.

。 Image Snapshot: Right clicking on the mosaic will bring up the option to capture an image snapshot of the color mosaic.

The steps of basic usage are indicated with numbers in the below screenshot.

1. Printer Icon: Prints the displayed image.

2. Overlapping Pages Icon: Copies the selected image.

3. Pat: Display motifs created e.g. in the Association Panel. Elements of the motif will be framed in yellow on the mosaic grid.

4. Abs: Absolute Value - Not used in this version.

5. All: Must be toggled on to display data, but not otherwise used in this version.

6. Ratio: Not used in this version.

7. Acce(ssion): Displays marker accession numbers to the right of the mosaic view.

8. Label: Displays marker annotations to the right of the mosaic.

9. 'All Arrays' and 'All Markers': When not selected, subsets of the markers and microarrays which have been activated are displayed. The selections are created from the activated Marker and Phenotype Panels. When selected, all markers and/or all arrays will be displayed.

10. Intensity slider: Widget enables user to change the color intensity of the mosaic display. The color of the displayed markers changes linearly with slider movement.

11. Gene Height: Widget lets user change the height of each marker/gene row to a required number of pixels.

12. Gene Width: Widget lets user to change the width of each column in the mosaic to a required number of pixels.
