Color-mosaic View

This is a utility to view Eisen plot views of a Microarray dataset. To view a dataset using Color-mosaic View, first select a dataset in the Project Panel and then select the Color Mosaic option in the main display pane at the top-right section of the interface. Measurements in the selected microarray dataset will be displayed as a two dimensional matrix with microarrays as columns and genes/markers as rows. Marker name is provided for each gene/marker on the right. A green-red color spectrum is used to indicate the relative magnitudes of the measurements. Back color indicates missing data.

This component contains some tools for users to manipulate the mosaic:

  • Marker Height:Widget lets user change the height of each marker/gene row to a required number of pixels.

  • Marker Width: Widget lets user to change the width of each column in the mosaic to a required number of pixels.

  • Accession: Button enables user to toggle marker names off and on.

  • Button with bulb: Widget enables user to toggle mouse-over information on the mosaic image.

  • Intensity slider: Widget enables user to change the color intensity of the mosaic display. The color of the displayed markers changes linearly with slider movement.

  • Right Click on mosaic image: This shows a pop-up menu with a 'Save Image' option. Selecting this option lets user save the mosaic image as a child of the currently selected dataset.

  • Clicking on mosaic: Clicking on any of the spots in the mosaic highlights the marker clicked in the Marker Panel.

  • 'Show Panels' and 'Show Phenotypes': Selecting these widgets enable the user to view marker and/or Microarray subselections. The selections are created from the activated Marker and Phenotype Panels.
