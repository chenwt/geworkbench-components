Position Histogram

The Position Histogram is located in the View Area in the upper right of the application. This component displays results from the Sequence Discovery Panel . This plots the positions of SPLASH motifs as a Position Histograms.

Basic Usage

The following summarizes the description and use of each option.

A. Plot Position: Displays the motifs in the Sequence Panel as they are selected in Pattern Discovery hit results. Each motif is represented by a specific color which is the same in both panels.

B. Save Graph: Stores the graph as a jpeg file.

C. Filter: Applies the filters of Avg/Peak and Abs/Rel.

D. ABS/Rel: Displays the fraction or absolute of total number of hits in the y axis. To apply, click Avg/Rel., Filter and Plot.

E. ABS/Peak: Alters step size to the sum of the p-value at every occurrence of the hit. To apply, click Abs/Peak, Filter and Plot.

F. Pairs and flex thr: These are not implemented.

H. Step: This is the average of the motif histogram across positions. The lower the step size the more fine-grained motif is displayed.

I. Display: Displays the graph which is stored when the graph is saved.

J. Support: The y axis is the number of sequence hits for that pattern.

K. Position: The x axis is the position on the sequence at which the hit occurs.

L. Legend: The legend for the sequences included in the position histogram.
